Relatório Técnico de Produção

Monitoramento e Coleta de Dados

O núcleo deste projeto de engenharia consiste em um sistema de telemetria para o processo de fermentação. Utilizamos um microcontrolador ESP32 para realizar a leitura constante da temperatura da cerveja durante as etapas críticas de produção.

Os dados coletados pelo sensor são enviados via Wi-Fi para uma planilha automatizada (ou banco de dados), permitindo o registro histórico e a análise do comportamento térmico do lote em tempo real. Isso garante a repetibilidade do processo e o controle de qualidade rigoroso exigido na produção industrial.

Automação com ESP32

Abaixo, o trecho de código responsável pela captura da temperatura através do sensor digital DS18B20:

// Requisição de temperatura ao sensor DS18B20
sensors.requestTemperatures(); 

// Leitura do valor em Celsius
float temp = sensors.getTempCByIndex(0);

// Lógica de envio para a base de dados
if (temp != DEVICE_DISCONNECTED_C) {
    enviarDadosParaPlanilha(temp); 
}

O Processo de Produção

Durante a brassagem e fermentação, o monitoramento nos permitiu ajustar o sistema de arrefecimento sempre que a temperatura divergia do setpoint estabelecido para a levedura utilizada.
